② 기기 설치 컨트롤러 · 허브 · 센서
협력사
① 컨트롤러 설치
  • 컨트롤러에 건전지를 삽입합니다.
  • 에어컨 제조사에 맞는 프로토콜을 선택합니다.
    • 삼성: C / LG: A / 캐리어: 프로토콜을 하나씩 모두 시도하여 제어 가능한 항목을 확인
  • 컨트롤러를 에어컨 가까이 가져가 블루투스 기반으로 제어 테스트를 진행합니다. (허브 등록 없이 단독 테스트)
  • 제어가 정상 확인되면 아래 기준에 따라 IR 수신부 방향으로 부착합니다.
    • IR 수신부와 평행이 되도록 설치
    • 필터 위에는 설치 불가
    • 실리콘을 적당량 도포
    • 낙하되지 않도록 견고하게 고정
  • 부착 완료 후 제어 테스트를 한 번 더 진행하여 정상 동작을 최종 확인합니다.
assets/ctrl-1.jpg · ctrl-2.jpg · ctrl-3.jpg
② 허브 설치
  • 컨트롤러 및 센서와의 수신 감도(dBm)를 확인하여 신호가 양호한 위치를 선정합니다.
  • 아래 기준을 준수하여 허브를 고정합니다.
    • 통신이 원활한 개방된 위치에 설치 (서랍 안 등 밀폐 공간 불가)
    • 흔들리지 않도록 견고하게 고정
assets/hub-1.jpg · hub-2.jpg · hub-3.jpg
③ 센서 설치
  • 센서에 건전지를 삽입합니다.
  • 아래 기준을 준수하여 부착합니다.
    • 매칭 에어컨 부근에 설치
    • 외기 영향, 에어컨 직풍, 철판, 발열기기 부근 피할 것
    • 센싱이 원활하도록 개방된 위치에 설치
    • 정렬을 맞춰 미관상 보기 좋게 설치
    • 좌식 공간: 바닥에서 약 1.2m / 입식 공간: 약 1.5m

  • 각 공간별로 설치한 기기의 MAC 주소를 입력하여 추가한 후 저장합니다.

설치 확인 — 사전 준비
설치된 에어컨 전체가 종료된 상태인지 확인한 후 진행합니다.
① 컨트롤러 제어 테스트
  • 파트너스앱 [장소 보기]에서 등록한 컨트롤러(에어컨)를 하나씩 선택합니다.
  • 냉방·송풍·난방 등 최소 3회 이상, 다양한 온도와 풍량 조합으로 제어 테스트를 진행합니다.
  • 선택한 기기와 실제 가동되는 에어컨이 일치하는지 확인합니다.
  • 기기 일치 확인 후 에어컨 명 옆 [>] → 오른쪽 상단 카메라 표시로 실시간 사진을 촬영합니다.
⚠ 주의 미리 촬영해 둔 갤러리 사진 업로드를 지양합니다. 제어 테스트 후 기기와 에어컨 매칭이 확인된 상태에서 실시간 촬영·업로드를 권장합니다.
촬영 가이드: 에어컨 위치와 설치 각도가 확인 가능하도록 촬영 (assets/ctrl-photo-1~3.jpg)
② 허브 사진 촬영
  • 허브의 MAC 주소와 촬영 기기가 일치하는지 확인합니다.
  • 설치 위치를 식별할 수 있도록 기기와 주변 공간이 함께 나오도록 촬영합니다.
assets/hub-photo-1~3.jpg
③ 센서 사진 촬영
  • 센서의 MAC 주소와 촬영 기기가 일치하는지 확인합니다.
  • 매칭된 컨트롤러와 함께 나오도록 촬영합니다.
assets/sensor-photo-1~3.jpg
④ 공유기 사진 촬영 (설치 시)
  • 씨드앤 공유기를 설치한 경우에만 진행합니다.
  • 공유기 스티커가 부착되어 있는지, 원공유기와 연결된 상태가 잘 보이도록 촬영합니다.

⑦ 도면 작성 및 등록 층별 배치 기록 · 파트너스앱 등록
협력사
기기 표기 방법
  • 허브 — [순번+H] 예: 1H → 2H (층 바뀌어도 이어서 증가, MAC 주소 12자리 기재)
  • 공유기 — [층+알파벳] 예: 1층 2대 → 1A, 1B / 2층 1대 → 2A
  • 컨트롤러 — [순번+C] 예: 1C, 2C (층 바뀌어도 이어서 증가)
  • 센서 — [순번+S] 예: 1S, 2S (층 바뀌어도 이어서 증가)
⚠ 번호 부여 규칙 층이 바뀌어도 번호는 이어서 증가합니다. 1층 마지막이 4C면 2층은 5C부터 시작합니다.
도면 작성 규칙
  • 고객이 주로 사용하는 에어컨에 앞번호, 사용 빈도가 낮거나 비상주 공간 기기는 끝번호로 부여합니다.
  • 파트너스 웹 등록 시 반드시 [도면 기입 번호] + [공간명(컨트롤러명)] 조합으로 등록합니다.
  • 에어컨 형태를 표기합니다. 예: 1way, 4way, 360형
  • 기둥, 장애물 등이 있을 경우 도면에 함께 표기합니다.
  • 방위(방향)를 전 층 동일한 기준으로 통일합니다.
    • 단층 매장: 출입문 기준
    • 복층 매장: 계단 기준
⚠ 방향 기준 혼용 금지 1층은 출입문 기준, 2층은 계단 기준처럼 층별로 기준을 달리하면 안 됩니다. 전 층 동일 기준을 사용합니다.
도면 등록 방법
  • 파트너스앱 [장소 보기] → 오른쪽 상단 [도면] 버튼을 누릅니다.
  • [+]를 누른 후 카메라 표시를 눌러 도면을 등록합니다.
  • 도면은 층별로 1개씩만 등록 가능하며, 모든 층마다 빠짐없이 등록합니다.

계정 생성 · 사용법 안내 앱 계정 생성 및 기능 설명
협력사
계정 생성
  • 파트너스앱 → 매장 [장소 보기] → [계정] 탭 선택합니다.
  • [+] → [계정 생성] 선택합니다.
  • 고객이 사용할 이메일 주소와 비밀번호를 입력합니다. 초기 비밀번호 통상 a1234567!
  • 역할(권한)을 선택합니다.
    • 마스터: 매장당 1명만 부여 가능, 매장 관리자에게 부여
    • 시니어 / 주니어: 생성 수 제한 없음
  • [생성]을 누르면 계정이 자동으로 해당 매장에 매칭됩니다.
assets/account-1~3.jpg
기존 계정 배정
  • [+] → [계정 배정] 선택합니다.
  • 이름 또는 이메일 주소 전체를 정확히 입력하여 검색합니다.
  • 역할(권한)을 선택한 후 [적용]합니다.
⚠ 검색 주의 일부 텍스트만 입력하면 결과가 나오지 않습니다. 이름 또는 이메일 주소 전체를 정확히 입력해야 합니다.
