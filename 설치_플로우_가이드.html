② 기기 설치컨트롤러 · 허브 · 센서
협력사
컨트롤러 설치
  • 컨트롤러에 건전지를 삽입합니다.
  • 에어컨 제조사에 맞는 프로토콜을 선택합니다.
    삼성: C / LG: A / 캐리어: 프로토콜을 하나씩 모두 시도하여 제어 가능한 항목 확인
  • 컨트롤러를 에어컨 가까이 가져가 블루투스 기반으로 제어 테스트를 진행합니다. (허브 등록 없이 단독 테스트)
    • 블루투스 제어 테스트 예시 (탭하여 영상 보기)
  • 제어한 값과 에어컨 LED 표시 값(온도, 풍량)이 일치하는지 확인합니다.
  • IR 수신부 방향으로 컨트롤러를 부착합니다.
    IR 수신부와 평행 설치 / 필터 위 설치 불가 / 실리콘 적당량 도포 / 낙하 방지 견고 고정
  • 부착 완료 후 제어 테스트를 한 번 더 진행하여 정상 동작을 최종 확인합니다.
허브 설치
  • 컨트롤러 및 센서와의 수신 감도(dBm)를 확인하여 신호가 양호한 위치를 선정합니다.
  • 허브를 고정합니다.
    통신이 원활한 개방된 위치에 설치 (서랍 안 등 밀폐 공간 불가) / 흔들리지 않도록 견고하게 고정
센서 설치
  • 센서에 건전지를 삽입합니다.
  • 설치 기준에 따라 매칭 에어컨 부근에 센서를 부착합니다.
    외기,직풍,철판,발열기기,출입구 부근 피할 것 / 개방된 위치에 설치 / 미관상 정렬 유지 / 좌식: 바닥에서 1.3m / 입식: 1.5m

⑦ 도면 작성 및 등록층별 배치 기록 · 파트너스앱 등록
협력사
기기 표기 방법
  • H (허브) — [순번+H] 예: 1H→2H, 층 바뀌어도 이어서 증가, MAC 12자리 기재
  • A (공유기) — [층+알파벳] 예: 1층 2대 → 1A, 1B
  • C (컨트롤러) — [순번+C] 예: 1C, 2C, 층 바뀌어도 이어서 증가
  • S (센서) — [순번+S] 예: 1S, 2S, 층 바뀌어도 이어서 증가
⚠ 번호 부여 규칙층이 바뀌어도 번호는 이어서 증가합니다. 1층 마지막이 4C면 2층은 5C부터 시작합니다.
도면 등록 체크리스트
  • 중요도 순으로 번호를 부여합니다.
    고객이 주로 사용하는 에어컨에 앞번호 / 비상주·사용 빈도 낮은 기기는 끝번호
  • 파트너스 웹 등록 시 [도면 기입 번호] + [공간명(컨트롤러명)] 조합으로 등록합니다.
  • 도면에 에어컨 형태를 표기합니다.
    예: 1way, 4way, 360형 / 기둥·장애물도 함께 표기
  • 방위(방향)를 전 층 동일 기준으로 통일합니다.
    단층: 출입문 기준 / 복층: 계단 기준 (층별 혼용 금지)
  • 파트너스앱 [장소 보기] → [도면] → [+] → 카메라 표시를 눌러 층별로 도면을 등록합니다. (층별 1개, 모든 층 빠짐없이)
⚠ 방향 기준 혼용 금지1층 출입문 기준, 2층 계단 기준처럼 층별로 기준을 달리하면 안 됩니다. 전 층 동일 기준을 사용합니다.

고객 사용법 안내
  • ① 제어 방법 — 앱에서 에어컨 직접 제어, 고객과 함께 실제 테스트 진행합니다.
    제어 방법 예시 (탭하여 영상 보기)
  • ② 스케줄 등록 방법 — 자동 운영 스케줄 등록 방법을 안내합니다.
    스케줄 등록 예시 (탭하여 영상 보기)
  • ③ 계정 안내 — 생성 계정 정보 전달, 초기 비밀번호 변경 방법을 안내합니다.
    계정 안내 예시 (탭하여 영상 보기)
  • ④ 스마트락 기능 설명 및 고객 사용 희망 여부 확인 후 씨드앤에 전달합니다.
    스마트락(캡스)이란?
    앱 상태와 실제 에어컨 상태를 항상 일치시켜주는 기능입니다. (예: 앱에서 OFF → 매장 에어컨도 자동 OFF)
    장점
    퇴근 후 리모컨 등으로 에어컨을 켜두고 가더라도, 설정된 스케줄 시간에 맞춰 에어컨이 자동 종료되어 낭비 전력을 차단합니다.
    작동 조건 & 주의사항
    · 전체 스케줄과 스마트락이 모두 설정되어야 작동합니다.
    · 리모컨으로 조작하더라도 강제로 앱 상태로 복원되므로, 온도 조절은 반드시 앱을 사용하도록 가이드 해주세요.
    스마트락(리프)이란?
    앱 설정값과 냉난방기 장치의 싱크를 고정하여 상태를 동기화하는 기능입니다.
    장점
    리모컨 오작동 및 실시간 제어 불일치 문제를 해결하여 최적화된 온도로 매장을 쾌적하게 유지합니다.
    작동 조건 & 주의사항
    · 별도의 스케줄 셋팅 조건이 요구되지 않습니다.
    · 강제 싱크가 맞춰지므로 온도 제어 시 스마트폰 앱 사용을 권장합니다.

    AI 모드란?
    설치된 온습도 센서가 공간의 온도·습도를 실시간 측정해 에어컨 설정 온도를 스스로 자동 조절합니다.
    장점
    공간의 과냉난방과 관리 사각지대를 인공지능이 알아서 밀착 제어하여, 실내 쾌적함을 최대화하면서 대폭 에너지를 절감합니다.
    AI 모드 작동 조건
    전체 스케줄과 스마트락이 정상 설정되어 있어야 합니다. (스케줄 범위 내에서 작동)
    AI 작동 시점
    AI가 판단하여 냉난방 작동이 필요할 때만 가동되므로, 스케줄 시작 후 실시간 작동에 약간의 시차가 발생할 수 있습니다.
    APP 제어 권고
    AI 모드 작동 상태에서 임시 수동 제어가 필요한 경우, 반드시 앱을 통해 제어하십시오. 앱 제어 패턴을 AI가 누적 학습하여 기호 환경을 제공합니다. (리모컨 제어 시 AI가 즉각 갱신하여 원복할 수 있습니다.)
    AI 모드란?
    온습도 센서 측정값 및 사용자 선호 온도 패턴 데이터를 활용하여 실시간 냉난방 설정을 자동 제어합니다.
    장점
    복잡한 스케줄 없이도 에어컨이 가동될 때 과도한 냉방/난방을 억제하고 낭비 전력을 대폭 감소시킵니다.
    작동 조건 & 주의사항
    · 스마트락 설정이 필수 조건입니다.
    · 켜져서 가동 중인 에어컨 디바이스를 대상으로 AI 제어 신호가 입력됩니다.
